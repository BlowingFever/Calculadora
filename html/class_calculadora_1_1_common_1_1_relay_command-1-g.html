Detailed Description

Generic ICommand implementation that accepts a strongly-typed parameter.

Template Parameters
TThe type of the command parameter.

Typical usage:

DigitCommand = new RelayCommand<string>(digit => OnDigit(digit));
RemoveCommand = new RelayCommand<ExpressionItem>(item => Remove(item));
