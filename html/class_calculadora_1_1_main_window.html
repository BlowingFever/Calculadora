Code-behind for MainWindow.xaml. Intercepts raw keyboard input at window level and forwards it to the active ViewModel's commands, preserving the MVVM separation.

MainWindow.

All presentation logic lives in MainViewModel. This file only handles the translation of physical key presses into commands.
