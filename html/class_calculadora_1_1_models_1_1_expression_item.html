Detailed Description

Represents a single mathematical expression entry in the graphing calculator UI.

Implements INotifyPropertyChanged directly (rather than inheriting ViewModelBase) so it can be data-bound inside an ItemsControl without pulling in the full ViewModel hierarchy.
