Detailed Description

Parameterless ICommand implementation backed by delegates. Allows commands to be defined concisely inside a ViewModel without a dedicated class.

Typical usage:

ClearCommand = new RelayCommand(() => OnClear());
SaveCommand = new RelayCommand(() => OnSave(), () => CanSave);
