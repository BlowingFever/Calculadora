Detailed Description

ViewModel de la calculadora gráfica. Gestiona la colección de expresiones matemáticas, evalúa cada una mediante mXparser y notifica a la vista cuándo debe redibujar el gráfico.

Flujo de uso:

  1. El usuario escribe expresiones en la lista lateral (Expressions).
  2. Cualquier cambio en texto, color o visibilidad dispara PlotRefreshRequested.
  3. El code-behind de la vista se suscribe al evento y llama a GetPlotData para cada expresión visible.
  4. ScottPlot renderiza los datos resultantes.

double [] Calculadora.ViewModels.GraphCalcViewModel.xs

Evalúa la expresión de item con mXparser dentro de los límites de los ejes visibles y devuelve 500 puntos de coordenadas (x, y) listos para pasar a ScottPlot.

Soporta tres formas de expresión:

FormaEjemplo
Función de Xsin(x) o y = x^2 + 1
Función de Yx = y^2 (curva horizontal)
Línea vertical constantex = 5

Si la expresión tiene sintaxis inválida o lanza una excepción durante la evaluación, el método devuelve null silenciosamente.

Parameters
itemEntrada de expresión a evaluar.
xMinLímite izquierdo del eje X visible.
xMaxLímite derecho del eje X visible.
yMinLímite inferior del eje Y visible.
yMaxLímite superior del eje Y visible.
Returns
Una tupla (xs, ys) con 500 elementos cada array, o null si la expresión está vacía, es inválida o falla en tiempo de evaluación.
