Detailed Description

Stateless arithmetic engine, fully decoupled from WPF and the MVVM infrastructure. Provides a single calculation operation that can be reused by any ViewModel.

Because this class holds no mutable state it is safe to create a single instance and share it across multiple ViewModels.
